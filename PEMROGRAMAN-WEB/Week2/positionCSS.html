position static;

Position static digunakan untuk mengatur element menjadi statos secara default. elemen akan mengikuti posisi normal secara default, elemen tidak dipengaruhi oleh properti seperti top, bottom, left, dan right.:

Elemen ini diatur menggunakan psosisi static;




position relative;

Sebuah element HTMLyang menggunakan position relative akan terletak pada posisi normal Mengatur properti atas, kanan, bawah, dan kiri dari elemn yang diposisikan dengan "relatif" akan membuatnya jauh dari posisi normalnya. Konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut.:

Elemen ini diatur menggunakan posisi : relative;




position fixed;

Sebuah element HTML yang di setting dengan position fixed memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scroll. Berlaku pengaturan left, bottom, top dan right pada position fixed:

Elemen ini diatur menggunakan posisi: fixed;















position: absolute;

Element HTML yang menggunakan position absolute akan diposisikan relatif dengan element lain yang mendahuluinya yang terdekat, bukan relatif terhadap laya secara normal.:

Elemen ini diatur menggunakan posisi: relative;
Elemen ini diatur menggunakan posisi: absolute;
